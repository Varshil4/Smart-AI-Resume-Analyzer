# 🚀 Smart AI Resume Analyzer

An AI-powered web application that helps job seekers optimize their resumes using intelligent analysis, ATS scoring, and data-driven insights.

🔗 **Live Repository:** https://github.com/Varshil4/Smart-AI-Resume-Analyzer

---

## 📌 Features

### 🔍 AI Resume Analyzer
- Analyze resumes (PDF/DOCX)
- Generate ATS score
- Get improvement suggestions
- Role-based skill recommendations

### 🧠 Smart Resume Builder
- Dynamic resume templates
- Easy form-based input
- Generate professional resumes

### 📊 Analytics Dashboard
- Track resume performance
- View ATS score trends
- Skill distribution insights

### 💼 Smart Job Search
- Trending skills analysis
- Job market insights

### 💬 Feedback System
- Collect user experience data
- Improve system based on feedback

## 🧠 Tech Stack

| Layer        | Technology |
|-------------|-----------|
| Frontend    | Streamlit |
| Backend     | Python |
| NLP         | SpaCy, NLTK |
| ML          | Scikit-learn |
| Database    | SQLite |
| Parsing     | pdfminer.six, python-docx |

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Varshil4/Smart-AI-Resume-Analyzer.git
cd Smart-AI-Resume-Analyzer

```

2️⃣ Create Virtual Environment
```bash 
python -m venv venv
venv\Scripts\activate   # For Windows
source venv/bin/activate   # For Mac/Linux
```

3️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```

▶️ Run the Application
```bash
streamlit run app.py
```
